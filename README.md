# Image Gallery Project

## Overview
This is a **responsive and interactive image gallery** built with **HTML, CSS, and vanilla JavaScript**.  
The gallery features a modern lightbox with smooth animations, keyboard navigation, and a professional design, making it ideal for portfolio or internship demonstration.

---

## Features

- **Responsive Grid Layout**: Automatically adjusts to different screen sizes.
- **Lightbox Preview**: Click an image to open it in a lightbox with smooth **fade and scale animations**.
- **Navigation**:  
  - Next / Previous arrows  
  - Keyboard support (`ArrowLeft`, `ArrowRight`, `ESC`)  
  - Click outside to close
- **Image Counter**: Displays the current image index and total number of images (e.g., `3 / 9`).
- **Mobile Friendly**: Fully responsive and touch-friendly layout.

---

## Technologies Used

- **HTML5**  
- **CSS3** (Grid, Flexbox, Transitions)  
- **JavaScript (ES6)**

---

## How to Run

1. Clone the repository or download the files.  
2. Open `index.html` in any modern web browser.  
3. Click on any image to view it in the lightbox and navigate using arrows or keyboard keys.
